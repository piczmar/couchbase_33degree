# Couchbase Developer Day Sample Code : Java

This project use Apache Maven ( http://maven.apache.org/ ) you can easily integrate it into your IDE or run the different sample from the command line.

Compile

	mvn clean package


Basic Connection:

	mvn -P01
	
	
Simple Storage

	mvn -P02
	
	
JSON Storage
	
	mvn -P03
	
	
Retrieve Data
	
	mvn -P04


Atomic Counter	

	mvn -P05


Non JSON Operations
	
	mvn -P06


CAS Operations
	
	mvn -P07


Durability Check
	
	mvn -P08

Views and Queries
	
	mvn -P09
